# 📷 Photo Compressor App

A minimalist, high-efficiency web application to compress images locally in your browser. Built entirely in Python using **Streamlit** and **Pillow (PIL)**.

🌐 **Live Demo:** [imagecomresszizizi.streamlit.app](https://imagecomresszizizi.streamlit.app/)

---

## ✨ Features

* **Instant Compression:** Reduce image file size significantly within seconds.
* **Format Support:** Accepts `.jpg`, `.jpeg`, and `.png` file formats.
* **Quality Control:** Interactive slider to balance between file size and image clarity.
* **Auto-Conversion:** Automatically optimizes transparent PNGs into efficient JPEGs for maximum compression.
* **Privacy Focused:** Image processing is handled securely within the memory buffer.

---

## 🛠️ Tech Stack

* **Core Language:** Python
* **Web Framework:** Streamlit
* **Image Processing:** Pillow (PIL)
